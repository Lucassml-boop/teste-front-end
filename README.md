# 🛍️ Vitrine de Produtos - Econverse

Este é um projeto desenvolvido em React e TypeScript que exibe uma vitrine de produtos consumindo dados de uma API JSON. Ele foi criado seguindo as diretrizes da Econverse para o desafio de Desenvolvedor Front-End Jr.
- ## 📌 **TEMPLATE DO FIGMA:** [Acesse aqui](https://www.figma.com/design/rWnzPeoxgynuNPsJjV0VmV/Teste-Front-End-Jr?node-id=0-1&p=f&t=I7ea48XBjdQ17WJY-0) (layout baseado em 1440px de largura).

## 🎯 Objetivo

Este projeto tem como objetivo exibir uma vitrine de produtos de maneira dinâmica e interativa. Ao clicar em um produto, um modal é aberto com mais informações sobre o item. O design segue fielmente o modelo fornecido no Figma.

## ⚙️ Tecnologias Utilizadas
- **Linguagem:** TypeScript
- **Framework:** React com TypeScript
- **Pré-processador CSS:** SASS
- **Gerenciador de Pacotes:** npm
- **Interação com Usuário:** Modal para detalhes dos produtos
- **Fonte de dados:** [JSON de Produtos](https://app.econverse.com.br/teste-front-end/junior/tecnologia/lista-produtos/produtos.json)

## 📌 Requisitos para Executar o Projeto

Para rodar esta aplicação, é necessário ter instalado:

- **Node.js:** [Instale aqui](https://nodejs.org/)
- **Git:** [Instale aqui](https://git-scm.com/)

Também é recomendado utilizar um editor de código como o Visual Studio Code.

## 🚀 Como Configurar e Rodar

```bash
# 1️⃣ Clone o repositório:
git clone https://github.com/Lucassml-boop/teste-front-end.git 
```

```bash
# 2️⃣ Navegue até o diretório do projeto:
cd teste-front-end
```

```bash
# 3️⃣ Instale as dependências:
npm install
```

```bash
# 4️⃣ Inicie o servidor de desenvolvimento:
npm run dev
```

📦 Build para Produção

```bash
# Para criar uma build de produção:
npm run build
```
## A aplicação estará disponível em http://localhost:5173.
